# Modal モーダル

画像の原寸表示用のモーダルダイアログ。

Jotaiを使った状態管理とハッシュベースのルーティングで動作。

## Props

<ArgTypes of={Modal} />

## 実装の詳細

### ハッシュルーティング

URL ハッシュによる表示制御：

- `#closeup`: モーダル表示状態
- ハッシュなし: モーダル非表示状態
